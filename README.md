# Sistema de Controle de Estoque para Pequenas Empresas

## Descrição do projeto

O Sistema de Controle de Estoque para Pequenas Empresas é uma aplicação web desenvolvida para organizar o cadastro, a entrada, a saída e a consulta de produtos. O projeto busca substituir controles realizados por cadernos, anotações e planilhas desatualizadas.

## Problema identificado

Muitas pequenas empresas ainda controlam seus produtos manualmente. Essa forma de trabalho pode causar erros na contagem, falta de mercadorias, compras duplicadas, perda de informações e dificuldade para acompanhar as movimentações realizadas.

## Objetivo

O objetivo do sistema é facilitar o controle do estoque, permitindo que usuários autorizados cadastrem produtos e fornecedores, registrem entradas e saídas, consultem a quantidade disponível e acompanhem o histórico das movimentações.

O sistema também deverá gerar alertas quando a quantidade de um produto ficar abaixo do estoque mínimo definido.

## Funcionalidades previstas

* Autenticação e login de usuários;
* Controle dos perfis administrador e operador;
* Cadastro de produtos;
* Cadastro de fornecedores;
* Registro de entradas no estoque;
* Registro de saídas no estoque;
* Consulta da quantidade atual dos produtos;
* Histórico de movimentações;
* Alerta de estoque mínimo;
* Relatório de produtos que precisam de reposição.
